Miller's Quest! – History and to-do list

0.9, 2005-10-09

  • This is the first actual release of Miller's Quest
  • Versioned 0.9 because a lot of the ideas that I came up with at the lateish point of development aren't there
  • The combat system is now present, but remains quite rudimentary
  • Healing system is present, but likewise pretty simple
  • Equipment stuff seems to work pretty well, apart of being pretty unbalanced
  • The game system remains a pretty ad-hoc thing; Perhaps a careful examination of the thing would be in order?

0.9.1, 2005-10-27

  • Combat fix from Josh Triplett
  • Command-line option parsing added
  • Rudimentary curses/terminfo-based display control (colors, bold, line-kill)
  • Fixed the item-selling decimal and negative-sale-price bug (I think it was just a reporting error.)

Features remaining unimplemented before 1.0
